Detailed Description

Класс главного окна приложения для демонстрации расчетов распределений и построения графиков.

Этот класс интегрирует взаимодействие с интерфейсом пользователя с вычислениями статистических распределений, обеспечивая динамическую визуализацию на основе пользовательского ввода.
