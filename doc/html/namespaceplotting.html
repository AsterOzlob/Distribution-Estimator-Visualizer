Строит графики распределений на предоставленном виджете.

Parameters
graph_widgetВиджет QWidget, на котором будут построены графики.
x_arrayОбщие данные для оси x для всех распределений.
y_truthДанные для оси y для графика истинного распределения.
y_edfДанные для оси y для графика эмпирической функции распределения.
y_kernelДанные для оси y для графика ядерного распределения.
Note
Используемые перья: RED для истинного распределения, BLUE для EDF и GREEN для ядерного распределения.
